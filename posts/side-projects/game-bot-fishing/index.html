Game Bot - Fishing

Game Bot - Fishing

Many years ago, i try to write a script to play video game automatically (only for testing).

Fishing is a mini game in WoW. This basic rule of this Fishing game is that player throw fishing rod(random position). And then player should wait the bait splash water (random time and no fix pattern for splash water because 3d) to quickly find the bait and click them at right time. (I don’t want to use way, such as replace pic or signal detect)

So it’s more complex to write script for 3D game. I try to use algorithm from opencv to find the bait position. But there is a problem how do i know the water is splash, it’s easy to mis-detect if i only use image recognition technology. And then i consider both audio(use pyaudio) and image recognition. When bait is ready, always play a very similar audio, so i use this to do double check to reach my goal(near no mis-detect).
