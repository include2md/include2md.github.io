Game Bot - Maze

Game Bot - Maze

Many years ago, I try to write a script to play mobile rpg video game automatically (only for testing).

Maze is a very important element in rpg game. When player try to pass maze, there is usually random encounter battle or event and also in the end of maze have a boss battle.

There are two interesting question

  1. How do we quickly train a shortest path for this maze?
  2. As stated above, how do we ensure the hero will finish shortest path?
    • Data structure design.
    • Random encounter battle cause wrong step for our hero and break calculation for shortest path.
