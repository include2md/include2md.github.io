3D Navigation System

This is my independent study project when i am a collage student.

At that time we build editor and let admin user can quickly build a scene with basic node info, such as building description, building node position and path graph.

And also display visual scene to client. client can navigate the scene like playing video game.

We also provide client many useful tool, such us filter, shortest path and animation display…etc.

This editor can also do something like interior design.
